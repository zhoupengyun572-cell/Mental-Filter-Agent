# 任务目标
利用大语言模型强大的角色扮演能力，生成具有统计学方差的青少年抑郁症状模拟数据集，用于系统计分模块的回归分析预演。

# 提示词主体
请为我生成 20 条模拟数据。必须严格满足以下统计学方差要求：
1. 【健康控制组】占 30%：完全没有心理问题，只是分享日常。所有 X 均为 0。
2. 【轻中度特异性组】占 50%：只触发 1 到 3 个特定的 X 维度，例如单纯的考试焦虑（X6, X10）、或者单纯的躯体化疲劳（X3, X4）。
3. 【重度及高危组】占 20%：触发 4 个以上维度，并且其中必须包含极少量触发 X11（自伤/自杀意念）的高危样本。
4. 【口语化要求】：文本必须极其符合当代 12-18 岁中国学生的真实网络语境，绝对不要像教科书一样机械地朗读症状。

请直接输出标准的 CSV 格式，包含表头 (ID, Text, X1~X11, Total_Score)。症状维度如果存在则赋值为 1，不存在则赋值为 0。
